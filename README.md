# Bruno Portfolio

Personal portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Run locally

1. Install dependencies:
   - `npm install`
2. Sync media assets from the original portfolio:
   - `npm run sync:assets`
3. Start development server:
   - `npm run dev`
4. Build for production:
   - `npm run build`

## Project structure

- `src/components` navigation and shared UI
- `src/pages` route-level pages
- `src/data/projects.ts` portfolio content and case study metadata
- `public` static assets (images, videos, PDF files)
